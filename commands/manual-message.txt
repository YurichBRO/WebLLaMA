Как развернуть бота, подобного этому?
Инструкция для Windows.

1. Установите Python3.X с https://www.python.org/downloads/
Обязательно поставьте галочку в "Добавить в PATH" при установке.

2. Установите Ollama с https://ollama.com/download

3. Запустите командную строку (cmd.exe)

4. Выполните в командной строке следующие команды:
Перейти в нужную папку
cd "Путь/к/папке/в/которой/будет/располагаться/бот"
Клонировать репозиторий
git clone "https://github.com/YurichBRO/WebLLaMA"
Подготовить виртуальное окружение
.\setup.bat

5. Создайте бота в помощью @BotFather в Telegram
Для этого нужно ему написать команду /newbot и следовать дальнейшим инструкциям (в том числе выбрать имя бота и его тег).
В конце вы получите токен бота

6. Создайте файл config.json
Введите в него следующее содержимое:
{
    "token": "ваш токен"
}

7. Создайте файл user.json
Введите в него следующее содержимое:
{}

8. Выполните в командной строке следующие команды:
Подготовить модель к запуску
ollama pull llama3.2
Запустить бота
python main.py

9. Напишите вашему боту
Найдите вашего бота в Telegram по тегу. Проверьте его функциональность.
